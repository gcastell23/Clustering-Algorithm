IGMC-Lite Pipeline

Uncertainty-Aware Shared Nearest Neighbor (SNN) Single-Cell Clustering

⚠️ Where We Started

Initially, our inductive pipeline suffered from scale distortion and structural limitations:

  • Metric Distortion: Artificial feature scaling blew up Euclidean spatial coordinates, distorting true clustering boundaries.
  • The Uncertainty Flood: A poorly calibrated validation split and low alpha boundary criteria caused the conformal prediction model to flag almost all cells (700+) as transitioning.
  • Data Pipeline Crashes: Biological gene filters crashed the program when synthetic control datasets with few variables were processed.

🚀 How It Evolved

We solved these critical bottlenecks by introducing clean biological and mathematical standards:

  • Dynamic Filtering: Developed dedicated biological preprocessing modules that automatically identify synthetic control data from complex scRNA-seq matrices.
  • Local Graph Tuning: Shifted from absolute spatial distance calculations to relative graph connectivity frameworks.
  • Calibrated Conformal sets: Tuned statistical tolerances to 85% confidence boundaries to cleanly highlight cell boundaries.

🛠️ The Four Pillars of IGMC-Lite

1. Density-Balanced SNN Graph

Instead of mapping flat spatial coordinates, we construct a Shared Nearest Neighbor (SNN) graph. Cells are mapped based on their shared neighbor intersections, which dynamically handles differing cell density profiles.

2. Normalized Spectral Manifold

We perform precomputed Spectral Partitioning on the SNN affinity graph. This handles complex, non-spherical biological manifolds that linear pipelines like K-Means miss.

3. Calibrated Split-Conformal Sets

By calibrating prediction non-conformity tolerances ($\alpha = 0.15$), we isolate cells that span multiple classes. This flags transitioning states selectively without visual noise.

4. Dual-Metric Verification

We monitor performance using the Adjusted Rand Index (ARI) for global partition quality and the Rare-Cell F1 Score to track how well we discover small cellular states.
